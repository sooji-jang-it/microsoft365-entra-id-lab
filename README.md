## Overview
This lab simulates a real-world enterprise identity and access management environment using Microsoft Entra ID and Microsoft 365 Admin Center. It focuses on user lifecycle management, security groups, role-based access control (RBAC), and cloud identity administration.

---

## Purpose
The purpose of this lab is to develop practical skills in Microsoft cloud identity management by simulating common IT administration tasks performed in enterprise environments, including user provisioning, group-based access control, and identity validation.

---

## Skills Covered
- Microsoft Entra ID (Azure Active Directory)
- User and group management
- Security groups and role-based access control (RBAC)
- Cloud identity and access management
- Microsoft 365 Admin Center administration
- Basic cloud security configuration
- Access validation and troubleshooting

---

## Environment
- Microsoft 365 Business Basic (Trial Tenant)
- Microsoft Entra ID (Azure Active Directory)
- Microsoft 365 Admin Center
- Cloud-based SaaS identity platform

---

## Lab Workflow

### 1. Tenant Setup
- Microsoft 365 tenant configured and verified
- Admin access established

### 2. User Management
- Users created in Microsoft Entra ID
- User properties configured

### 3. Security Groups
- Security groups created
- Groups used for access control and role assignment

### 4. Group Membership
- Users assigned to appropriate security groups
- Access policies applied through group membership

### 5. Access Validation
- Verified user access based on group membership
- Confirmed role-based permissions function correctly

---

## Outcome
The lab successfully demonstrates core identity and access management operations in a cloud-based Microsoft environment, including user provisioning, group-based access control, and validation of security configurations.
